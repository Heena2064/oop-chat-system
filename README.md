# OOP Chat System

A simple console-based chat system built using Python and Object-Oriented Programming (OOP) concepts.

## Features

- Create users
- Create a chat room
- Join and leave the chat room
- Prevent duplicate user joins
- Allow only joined users to send messages
- Store chat message history
- Display active users
- Add timestamps to messages

## OOP Structure

The project uses three main classes:

### User
Represents a user in the chat system and stores the user's name.

### Message
Represents a chat message and stores:
- Sender
- Message text
- Timestamp

### ChatRoom
Manages:
- Chat room name
- Users in the room
- Message history
- Joining and leaving users
- Sending messages
- Displaying chat history
- Displaying active users

## How to Run

Make sure Python is installed on your system.

Open the project folder in a terminal and run:

```bash
python chat_system.py
```

- Constructors (`__init__`)
